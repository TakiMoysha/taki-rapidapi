## Installation

```bash
uv sync
uv tool install maturin
maturin develop --uv
./venv/bin/maturin_import_hook site install
maturin_import_hook site install
uv tool install maturin
uv run pip install maturin_import_hook
uv run maturin_import_hook site install
```
